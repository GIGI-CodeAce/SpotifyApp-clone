<h4>Here're some of the project's best features</h4>

*   Album name  and cover image direct site changing
*   Really easy data changing and switching
*   Snippet of code from album cover image changer:

```javascript
// ...
    <div className="relative bg-gradient-to-b from-slate-700 via-black to-black h-80 overflow-hidden">

            backgroundPosition: 'center',
```
